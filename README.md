# 🎓 Saathi — Academic Intelligence

> **Elevate your learning with a professional, industry-standard study companion designed for high-performance students.**

![Saathi Header](https://raw.githubusercontent.com/lucide-icons/lucide/main/icons/brain.svg)

Saathi is a sophisticated React-based study companion that replaces generic "AI-styled" clutter with a clean, high-contrast **White & Royal Blue** design system. It focuses on productivity, recall optimization, and task management through a streamlined, professional dashboard.

---

## 🚀 Key Features

- **📊 Comprehensive Dashboard**: Get a birds-eye view of your academic progress with real-time productivity stats and task queues.
- **📅 Smart Revision Planner**: An interactive calendar system that intelligently tracks and schedules your revision sessions to optimize long-term recall.
- **✅ Professional Task Management**: A rock-solid filtering and sorting system for subjects, topics, and priorities—designed to stay stable even with complex data.
- **🧠 Academic Intelligence**: Integrated study assistant tools (AI-powered) designed to help you break down complex subjects without the visual distraction of typical AI interfaces.
- **📱 Responsive Layout**: A professional sidebar navigation and header system that works seamlessly across desktop and mobile.

## 🛠️ Technology Stack

- **Frontend**: [React.js](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (Solid Professional Theme)
- **Routing**: [React Router v6](https://reactrouter.com/) (Outlet-based Architecture)
- **Animations**: [Framer Motion](https://www.framer.com/motion/) (Micro-interactions)
- **Icons**: [Lucide React](https://lucide.dev/)
- **State Management**: React Context API (Study & Auth)
- **Notifications**: React Toastify

## 📂 Project Structure

```text
src/
├── components/     # Reusable UI components (RevisionList, TaskCard, etc.)
├── context/        # Global state management (Auth, Study Data)
├── pages/          # Full-page views (Dashboard, Tasks, Revision, AI Tools)
├── index.css       # Core design system and 'Industry' utility classes
└── App.jsx         # Routing engine and Layout definition
```

## ⚙️ Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd study-companion
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

## 🎨 Design Philosophy

Saathi follows a **"Professional Man"** design philosophy:
- **Zero Fluff**: No neon glows, pulsing animations, or glassmorphism.
- **High Contrast**: Dark slate text on solid white backgrounds for maximum legibility.
- **Solid Elevations**: Uses professional shadows and borders to define hierarchy.
- **Academic Focus**: Terminology and icons curated specifically for the academic environment.
